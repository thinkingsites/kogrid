Documentation

  • Options

    • url

    • columns

      • title the header for the column
      • template - a knockout template
      • key
      • css - pass css classes as a string into a column
      • style - pass in fixed styles into a column
      • visible - boolean on whether a column is hidden or not
      • data - extra data to pass into each grid cell, available as part of the template $data. functions can be added here. the 'this' parameter is the row data item
    • rows - if url is not used, the row data can be passed in directly
    • total - if url is not used the total number of rows can be passed in directly
    • renderComplete - will be fired after the grid has been rendered, will fire any time rows are reloaded
